Sesión 4
Github

Esta semana nos enseñaron a subir nuestras paginas web usando Github, esto abarcó desde la inicialización de un repositorio local hasta la creación de un repositorio remoto en GitHub. Se nos guió a través de los pasos para agregar, confirmar y enviar nuestros archivos al repositorio remoto, permitiéndonos comprender cómo funciona el flujo de trabajo colaborativo en un entorno de desarrollo compartido.
